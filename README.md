# 🛰️ DARKRAY: Deep-Gravity SIGINT & Network Mapper
**Mapeamento de Ativos via Entrelaçamento de Dados e Grade Astrométrica de Hardware**

O **DARKRAY** (DGNM v1.1) é uma ferramenta de inteligência cibernética avançada projetada para realizar o reconhecimento passivo e a modelagem matemática de alvos digitais. Ao integrar a biblioteca `Astropy`, a ferramenta projeta o endereço IPv4 de um alvo em um plano de coordenadas celestes, permitindo a extração de assinaturas determinísticas que vinculam o hardware lógico a um ponto fixo em uma grade de simulação gravitacional.

---

## 🔍 Pilares da Inteligência Cyber

### 1. Vetorização de Ativos (Astrometria Computacional)
Diferente do mapeamento geográfico comum, o DARKRAY utiliza a **Ascensão Reta (RA)** e a **Declinação (Dec)** para criar um identificador único de hardware. 
* **RA (Profundidade):** Representa a posição horizontal do ativo na grade de simulação.
* **Dec (Gravidade):** Representa a "massa lógica" ou a influência do host na rede, normalizada por funções senoidais para garantir integridade física (entre -90° e 90°).

### 2. Análise de Canal Lateral Acústico (Acoustic SIGINT)
A ferramenta converte a telemetria de rede em **Consoantes de Pontos Audíveis**. Isso permite que o analista de inteligência:
* Identifique assinaturas de rede através de padrões sonoros (`.mp3`).
* Detecte variações sutis em infraestruturas distribuídas que seriam invisíveis em logs de texto puro.
* Crie um arquivo histórico de áudio para comparação de "ruído de base" de servidores alvo.

### 3. Entrelaçamento Eletromagnético (C2 Mapping)
O script opera em uma lógica de comando e controle (C2), gerando logs críticos diretamente na linha de terminal. O processo de resolução DNS é tratado como um "entrelaçamento", onde a informação do domínio é colapsada em um estado fixo (IPv4), permitindo a exploração da "profundidade" do hardware remoto.

## 🛠️ Especificações Técnicas

| Módulo | Função | Descrição de Inteligência |
| :--- | :--- | :--- |

| **Astropy Engine** | Gravity-Grid Projection | Conversão de octetos em vetores de SkyCoord (ICRS). |
| **Audio Synth** | Acoustic Sonification | Geração de relatórios auditivos em formato comprimido (.mp3). |
| **Report Engine** | Forensic Logging | Documentação técnica em CSV para ingestão em ferramentas de SIEM. |

---

## 📈 Fluxo de Trabalho de Exploração

1.  **Reconhecimento:** O operador insere o domínio alvo.
2.  **Colapso de Dados:** O IPv4 é processado e somado, servindo como semente para a grade astrométrica.
3.  **Cálculo Gravitacional:** O script aplica funções trigonométricas para garantir que os dados respeitem as leis da física (astrodinâmica computacional).
4.  **Exfiltração:** São gerados dois artefatos: um log forense (`.csv`) e uma assinatura acústica (`.mp3`).

## 🚀 Instalação e Execução

Para iniciar a atividade de inteligência, certifique-se de possuir as dependências instaladas:

```bash
pip install astropy requests gtts numpy
```

Execução:
```bash
python darkray.py
```
