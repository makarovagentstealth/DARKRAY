# 🌌 Deep-Gravity Network Mapper (DGNM) v1.0
**Exploração de Profundidade de Hardware e Entrelaçamento de Dados via Astrometria**

O **DGNM** é uma ferramenta experimental de inteligência cibernética (OSINT) e segurança avançada que transcende o mapeamento de rede tradicional. Ela utiliza a biblioteca `Astropy` para correlacionar ativos digitais (IPv4) com coordenadas físicas celestes, criando uma "assinatura gravitacional" única para cada host.

## 🛡️ Aplicações em Segurança Avançada

Esta ferramenta foi desenhada para analistas que buscam uma camada extra de abstração e segurança física na identificação de infraestruturas:

1.  **Assinatura de Hardware Determinística:** Ao converter um IPv4 em uma grade de coordenadas `SkyCoord`, a ferramenta gera um identificador que não depende apenas de registros WHOIS, mas de uma matriz matemática fixa baseada na topologia da rede.
2.  **Monitoramento Acústico (Audible Points):** A conversão de dados em arquivos `.mp3` permite que operadores de segurança identifiquem anomalias na infraestrutura através de padrões rítmicos. Mudanças sutis na "gravidade" do domínio resultam em variações nas consoantes acústicas geradas.
3.  **Mapeamento de Profundidade:** Utiliza conceitos de física teórica para demonstrar a "distância" lógica entre o hardware do atacante/alvo e a grade de exploração, auxiliando em processos de atribuição de ameaças.

## 🚀 Funcionalidades Principais

* **Entrelaçamento Eletromagnético de Rede:** Mapeia domínios DNS para IPv4 com logs em tempo real no terminal (C2 Style).
* **Astrometria Computacional:** Integração com `Astropy` para cálculo de Ascensão Reta (RA) e Declinação (Dec) baseados em pacotes de rede.
* **Relatórios Críticos:** Geração de logs `.csv` detalhando a física por trás de cada ponto de hardware explorado.
* **Saída Acústica:** Síntese de voz e padrões de frequência salvos diretamente em `.mp3` para análise forense auditiva.
